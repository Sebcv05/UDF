Physics Implementation:
-----------------------
Song RPE (Equation 4):
  R̈ = [P_sat - P_∞ + (2σ/R₀+P_r0)·(R₀/R)³ - 2σ/R - 4μ·Ṙ/R - 4κ·Ṙ/R²] / (ρ_m·R) - (3/2)·Ṙ²/R

Parameters:
  - P_r0 = 1.0e6 Pa (residual gas pressure)
  - κ = 0.0 (surface viscosity - negligible)
  - R_spec = 488.2 J/(kg·K) for NH3
  - Void target = 0.99

Data Flow:
----------
INPUT (from parcel_cloud[p_idx]):
  - r_bubble, v_bubble       (current state)
  - r_bubble_0, r_drop_0     (initial conditions)
  - temp, density, viscosity (properties)
  - surf_ten                 (surface tension)

COMPUTE:
  - P_sat from temperature (isothermal)
  - Void fraction ε
  - Mixture density ρ_m
  - Acceleration R̈

UPDATE (single timestep):
  - Rdot += Rddot * dt_sub
  - R += Rdot * dt_sub

OUTPUT (to parcel_cloud[p_idx]):
  - r_bubble (updated)
  - v_bubble (updated)
  - temp (UNCHANGED - key difference from thermal model)

Safety Checks:
--------------
1. Droplet radius (Ro) too small → reset
2. Initial bubble radius (R0) too small → reset
3. Not superheated (P_sat ≤ P_amb) → reset
4. Void fraction ≥ 0.99 → reset (complete)
5. Bubble collapse (Rdot < 0) → reset
6. Bubble at droplet edge (R > 0.999·Ro) → cap
